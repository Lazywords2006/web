# 项目结构说明

## 📁 核心目录

```
网络验证/
├── server/                    # 服务器端（核心）
│   ├── main.go               # 服务器主程序
│   ├── handlers/             # API 处理器
│   ├── database/             # 数据库操作
│   ├── models/               # 数据模型
│   ├── utils/                # 工具函数
│   └── frontend/             # Web 管理界面
│
├── docs/                      # 文档目录
│   ├── 集成到EXE的完整指南.md
│   ├── Python_GUI_使用说明.md
│   └── 其他文档...
│
├── examples/                  # 示例代码
│   ├── python_gui_example.py
│   └── requirements.txt
│
├── auth/                      # 客户端认证模块
├── hwid/                      # 硬件ID生成模块
├── heartbeat/                 # 心跳监控模块
│
├── deploy/                    # 部署配置
├── README.md                  # 主文档
└── go.mod                     # Go 模块定义
```

## 🔧 开发说明

### 客户端模块（可复用）
- `auth/` - 许可证激活和验证
- `hwid/` - 跨平台硬件ID生成
- `heartbeat/` - 后台心跳监控

### 服务器部署
- `deploy/` - Docker 和云平台部署配置
- `docker-compose.yml` - 本地 Docker 部署
