# 示例代码

本目录包含各种集成示例和工具。

## 📦 lzy_zte 项目集成示例

### 目录：[lzy_zte/](lzy_zte/)

完整的许可证验证集成示例，包含：

- **launcher_cli.py** - 命令行版启动器（推荐）
- **launcher_wrapper.py** - GUI 版启动器
- **启动器_命令行.command** - 一键启动脚本（命令行）
- **启动器.command** - 一键启动脚本（GUI）

### 使用方法

详见：
- [集成说明](../docs/lzy_zte_integration/集成说明_lzy_zte.md)
- [快速开始](../docs/lzy_zte_integration/快速开始_lzy_zte.txt)

## 🚀 快速开始

```bash
# 进入示例目录
cd lzy_zte

# 运行命令行版（推荐）
python3.11 launcher_cli.py

# 或双击运行
# 启动器_命令行.command
```
