# license_launcher 文件说明

## 📁 目录结构

```
license_launcher/
├── 📄 核心文件
│   ├── launcher.py                 # 启动器源代码
│   ├── launcher_config.json        # 配置文件
│   └── lzy_zte_12.10.exe          # 要打包的程序
│
├── 🛠️ 打包脚本
│   ├── build_with_program.bat     # Windows 打包
│   └── build_with_program.sh      # macOS/Linux 打包
│
├── 📦 打包结果
│   └── dist/
│       ├── lzy_zte_许可证验证     # 主程序 (35MB)
│       ├── lzy_zte_12.10.exe      # 运行后自动释放
│       └── .license                # 许可证缓存
│
├── 📚 文档
│   ├── README.md                   # 完整文档
│   ├── QUICKSTART.md               # 快速开始
│   ├── lzy_zte_打包流程.md        # 打包流程
│   ├── 生成Windows_EXE指南.md     # Windows 指南
│   ├── 文件释放机制说明.md        # 技术细节
│   └── 项目完成总结.md            # 项目总结
│
└── 📋 其他
    └── requirements.txt            # Python 依赖
```

## 🎯 快速开始

### 打包程序
```bash
# macOS/Linux
./build_with_program.sh

# Windows
build_with_program.bat
```

### 分发给用户
只需复制:
```
dist/lzy_zte_许可证验证
```

### 用户使用
```bash
# 首次运行
./lzy_zte_许可证验证
# 输入许可证密钥

# 后续运行
./lzy_zte_许可证验证
# 自动验证并启动
```
